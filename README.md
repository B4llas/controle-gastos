\# 💰 Controle de Gastos



Aplicação web para controle financeiro pessoal com visualização de dados e organização por categorias.



\---



\## 🔗 Demonstração

👉 https://b4llas.github.io/controle-gastos/



\---



\## 📸 Preview



!\[Preview do projeto](./preview.png)



\---



\## ⚙️ Funcionalidades



\- Adicionar gastos

\- Categorias (alimentação, transporte, lazer, outros)

\- Filtro por mês

\- Resumo automático de gastos

\- Gráficos (pizza, barra e linha)

\- Persistência com localStorage

\- Edição e remoção de gastos



\---



\## 🧠 Tecnologias



\- HTML5

\- CSS3

\- JavaScript (ES Modules)

\- Chart.js

\- LocalStorage



\---



\## 📌 Objetivo



Projeto desenvolvido para praticar:



\- Manipulação de DOM

\- Organização de código modular

\- Lógica de dados

\- Visualização de informações

\- Criação de dashboards
